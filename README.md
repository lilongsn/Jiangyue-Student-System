# 简予 JIANGYUE - 学生管理系统

**简予 (JIANGYUE)** 是一款专为校园环境设计的轻量级学生信息管理系统。系统采用极简的“毛玻璃”视觉风格，提供直观的数据看板、学生档案管理及成绩追踪功能。

-----

## 📖 项目介绍

本项目旨在解决传统管理系统界面沉重、操作复杂的问题。通过响应式布局与扁平化设计，简予提供了一个既美观又实用的管理终端，适用于中小型教育机构或班级内部管理。

### 核心亮点

  * **极简视觉**：基于 Tailwind CSS 的高级毛玻璃 (Glassmorphism) 效果。
  * **实时看板**：集成 Chart.js，自动统计专业分布与男女比例。
  * **档案闭环**：支持学生信息的录入、检索、修改、逻辑删除及回收站恢复。
  * **安全认证**：内置用户登录鉴权，保护学生隐私数据。

-----

## 🏗 项目架构

项目采用经典的 **MVC (Model-View-Controller)** 模式开发，前端利用 Jinja2 模板引擎进行动态渲染。

```text
jianyu_system/
├── app.py              # 控制器：核心业务逻辑与路由跳转
├── seed_data.py        # 辅助工具：初始化管理员账号与模拟数据
├── requirements.txt     # 依赖清单：项目运行所需的库
├── edu_pro.db          # 数据库：SQLite 文件 (运行后生成)
└── templates/          # 视图层：HTML 模板
    ├── base.html       # 全局母版 (导航、样式头)
    ├── dashboard.html  # 数据可视化页面
    ├── list.html       # 学生名册列表
    ├── form.html       # 学生录入/修改表单
    ├── grades.html     # 成绩中心
    └── login.html      # 系统登录页
```

## 🚀 运行指南 (虚拟环境)

为确保项目依赖不污染全局环境，建议使用 Python 虚拟环境运行：

### 1\. 创建并激活虚拟环境

在项目根目录下运行：

```bash
# 创建虚拟环境 (文件夹名为 venv)
python -m venv venv

# 激活虚拟环境 (Windows)
venv\Scripts\activate

# 激活虚拟环境 (macOS/Linux)
source venv/bin/activate
```

### 2\. 安装依赖

```bash
pip install -r requirements.txt
```

### 3\. 初始化数据

运行数据种子脚本，创建 `admin` 账号并生成 100 条模拟学生数据：

```bash
python seed_data.py
```

  * **默认登录账号**：`admin`
  * **默认登录密码**：`123456`

### 4\. 启动服务

```bash
python app.py
```

访问浏览器：`http://127.0.0.1:5000`

## 🛠 功能示例

### 📊 数据看板

系统首页自动统计在校学生总数及平均分。通过环形图和条形图实时展示各专业的生源比例及性别均衡度。

### 👥 学生管理

  * **检索功能**：支持通过姓名或学号快速筛选学生。
  * **名册展示**：清晰列出学号、姓名、专业班级信息。
  * **档案编辑**：支持一键跳转编辑表单，实时更新数据库信息。

### 📝 录入与修改

通过 `form.html` 提供标准的交互表单。系统会自动判断是“新增”还是“修改”现有档案，并对学号唯一性进行校验。

### 🎓 成绩中心

聚合展示全校/全班学生的学分绩点与成绩分布情况，支持快速预览综合评分。
